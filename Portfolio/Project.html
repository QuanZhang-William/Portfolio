Step Counter Navigation App (Android)
  • Used finite state machine, low-pass filter to filter out noise from sensor and to improve accuracy
  • The app contains step counter feature which uses acceleration sensor to detect change of grativity chnage in movement
  • Read a given map (in special format provided by professor). Utilized Android magnetic sensor to detect orientation. User is able to set a destination and the shortest path will be displayed. User's position is displayed on map as well.
Driver of Speakers in FAT file system
  • C Programming in FPGA Board
  • Read a sound file (.wav file) in SD Card; Buffer the data and send to a connected speaker.
  • The playback mode can be controlled in FPGA board (reverse, double speed, delay, half speed, normal)
  • Applied lower-level knowledge in software and hardware programming including advance data structer, concept of polling and interrupt and FPGA programming
Arduino Parking Assistance System
  • A small Arduino project using distance sensor, LED display screen and potentiometer
  • This project uses distance sensor to detect the distance between obstacle and the system. The result is displayed on a LED display screen. A red warning LED will light up if the obstacle is too close to the system. User is also able to adjust the lightness of warning LED via a potentiometer.
  • Also implemented a Java program to talk to arduino via serial connection. Readings from arduino is streamed to the Java program through the connection which is then directed and stored in database by using JDBC.
